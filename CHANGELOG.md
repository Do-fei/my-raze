# Changelog

本文件记录 My Raze 项目的所有重要变更。格式遵循 [Keep a Changelog](https://keepachangelog.com/zh-CN/1.1.0/)。

## [3.0] - 2026-02-21

### 新增
- 亲密度等级系统：10 级等级、经验值积累、专属称呼
- 升级动画：爱心粒子特效 + 解锁内容展示
- 防刷分机制：每日上限 + 冷却时间 + 多维度加分
- 对话/语音/自拍/编辑资料均可获得经验值
- 39 个亲密度相关测试用例
